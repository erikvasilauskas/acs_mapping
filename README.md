This repository contains materials, code examples, and resources from a public presentation on working with and mapping American Community Survey (ACS) data in R.

You can request a [Census API key here.](https://api.census.gov/data/key_signup.html)

The examples focus on practical, reproducible workflows using:

- tidycensus
- tigris
- tidyverse
- ggplot2

The goal is to help users access Census data, create informative maps, and build confidence working with spatial data in R.

## Topics Covered

- Accessing ACS data with `tidycensus`
- Working with Census geographies
- Downloading shapefiles with `tigris`
- Joining data and geometry
- Creating choropleth maps
- Styling maps with `ggplot2`
- Exploring demographic and socioeconomic patterns
